# Multiple projects made on Python for venv-pip management at platzi

Follow the instructions to execute each project

## Chart Project

```sh
cd game
python3 main.py
```
## Charts Project

```sh
cd charts
python3 main.py
```

## App Project

```sh
git clone
cd app
python3 -m venv env
source env/bin/activate
pip3 install -r requirements.txt
python3 main.py
```

## Web-server Project

```sh
git clone
cd web-server
python3 -m venv env
source env/bin/activate
pip3 install -r requirements.txt
python3 main.py
```
